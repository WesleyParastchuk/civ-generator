# Civ Generator

Tela inicial tematica para sorteador de partidas de Civilization VI, criada com
Next.js (App Router + TypeScript + Tailwind CSS v4).

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000` no navegador.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Deploy na Vercel

1. Suba o projeto para um repositorio no GitHub.
2. Importe o repositorio na Vercel.
3. A Vercel detecta automaticamente Next.js e usa o build padrao.
4. Em cada push na branch principal, o deploy e atualizado.

Nao e necessaria configuracao extra para essa etapa inicial.
